How to use a periodic table

This section will teach you how to read a periodic table if you don't know already.

If you already know, click here for site-specific features.


At any point in this page, if you need more information on a certain concept, click on a the link for a Wikipedia article that explains it in much more detail.

Overview

A periodic table is a chart used for looking up information about elements, i.e. types of atoms. The table is set up logically - similar elements are close to eachother, and patterns in the table can be used to instantly find information. Periodic tables are used extensively in nearly all chemistry labs - amateur to professional.

The Element

The periodic table is made up of elements. On almost all tables, including this one, elements are displayed as boxes with information inside them.

Here is an enalarged example, with a side-by-side explanation of what each thing means:

Element With Labels

  • Atomic Number (Top-Left) --- The number of protons in that element.
  • Atomic Weight (Top-Right) --- The average mass of isotopes (variants) of a natural sample of the element.
  • Atomic Symbol (Center) --- A one or two letter abbreviation of the Element Name.
  • Element Name (Bottom) --- The scientific name of the element.

The Table

The table orders elements into rows (periods) and columns (groups).

Each element has electrons, which form themselves into layers. Periods are split up by the number of layers an element has: Hydrogen has 1 layer, so its in the top period, Lithium has 2, so it's in the next period down, etc.

NOTE: Lanthanides and Actinides are NOT seperate rows - they expand out from the "Lanthanides:" and "Actinides:" labels on the table. They are displayed below the rest of the table to make the table easier to read.

Groups are split up roughly by the number of electrons the outer layer has: Hydrogen has 1 electron, so it's in the leftmost layer, Helium has 2, which fills the layer, so it's in the rightmost period, etc.

Unique Features

Coloring by type

This feature is common in online periodic tables but not every site has it. An element's color corresponds to its type - a group of elements with similar properties.

Here's a list:

  • --- Hydrogen (has its own type)
  • --- Alkali Metals
  • --- Alkaline Earth Metals
  • --- Transition Metals
  • --- Post-Transition Metals
  • --- Metalloids
  • --- Reactive Nonmetals (excluding hydrogen)
  • --- Halogens
  • --- Noble Gases
  • --- Lanthanides
  • --- Actinides
  • --- Uncategoized (currently represents no elements)

More Info page

Clicking on an element takes you to a "more info" page, showing the element in greater detail, as well as additional information about the selected element.

Here's what it looks like:

  • Group --- What column an element is in.
  • Period --- What row an element is in.
  • Electronegativity --- The strength that the element attracts free electrons.
  • Type --- Groups similar elements together. Used here colliquially, generally following IUPAC guidelines.

Search

The search bar allows you to type the name of an element to instantly see its More Info page.

Select the search bar to start typing in it, to search, press Enter on your keyboard or click the button.

NOTE: You must spell the element name exactly right; there's no autocorrect. Capitalization doesn't matter.
